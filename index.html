How To Play

  1. Press any key on the keyboard to start.
  2. Now, click on the blinked colour(e.g blue).
  3. Next, the game shows the next colour (red), the user has to remember the sequence is blue, red and click on all the colours from the very beginning along with the blinked colour in each level.
  4. If the user messes up the sequence, then the game ends.
